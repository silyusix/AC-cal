# 自动控制原理计算器前端

这是一个基于 React 和 Vite 构建的自动控制原理计算器前端应用。

## 运行项目

1.  **安装依赖**

    在项目根目录下运行以下命令，安装所有工作区依赖：

    ```bash
    npm install
    ```

2.  **启动开发服务器**

    在项目根目录下运行以下命令，启动前端开发服务器：

    ```bash
    npm run dev --workspace=frontend
    ```

    或者，您可以直接进入 `frontend` 目录并运行 `npm run dev`：

    ```bash
    cd frontend
    npm run dev
    ```

    应用将在 `http://localhost:5173` (或类似地址) 启动。

## 使用说明

### 标准计算器

*   点击数字按钮输入数字。
*   点击运算符按钮进行加、减、乘、除运算。
*   点击 `.` 按钮输入小数点。
*   点击 `C` 按钮清空显示。
*   点击 `=` 按钮获取结果。

### 自动控制原理计算器

1.  **输入传递函数**
    *   在“分子系数”和“分母系数”输入框中，输入传递函数的系数，用逗号 `,` 分隔。例如，对于 `s + 1`，分子系数为 `1,1`；对于 `s^2 + 2s + 1`，分母系数为 `1,2,1`。
2.  **点击“分析”按钮**
    *   系统将尝试解析输入的系数并进行分析。目前，分析结果将以 JSON 字符串形式显示。
